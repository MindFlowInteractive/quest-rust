## 🚀 Getting Started

### Prerequisites
- [Rust](https://www.rust-lang.org/tools/install) (latest stable recommended)
- Cargo (comes with Rust)

Verify installation:
```bash
rustc --version
cargo --version

# Initialize / Build

# Clone the repository and build:

git clone https://github.com/MindFlowInteractive/quest-rust.git
cd quest-rust
cargo build

# Run
cargo run

# Test
cargo test
